# mod-404-reliability-cost-incident: Reliability, Cost & Incident Response

> Scaffolded by `aicg org execute-plan`. Lecture chapters and exercise content are authored on subsequent autonomous cycles.

**Estimated effort:** 12 hours

## Learning objectives

- Define SLOs/SLIs for agentic systems and alert on quality drift, not just uptime
- Build cost controls and budgets into running systems
- Run incident response for agent failures (bad outputs, runaway loops, tool misuse)
- Write postmortems and drive durable fixes

## Structure

- `01-…md` … `0N-…md`: lecture chapters.
- `exercises/`: per-exercise prompts.
- `labs/`: long-form hands-on labs.
- `quizzes/`: knowledge checks.
- `resources.md`: external references.

# mod-401-agent-systems-in-practice: Implementing Agent Systems from an Architecture

> Scaffolded by `aicg org execute-plan`. Lecture chapters and exercise content are authored on subsequent autonomous cycles.

**Estimated effort:** 14 hours

## Learning objectives

- Turn a reference architecture into a production implementation; own a subsystem end to end
- Make and document implementation-level tradeoffs (framework choice, state, tool boundaries)
- Refactor a prototype agent into a maintainable, testable codebase
- Drive build-vs-buy and complexity decisions at the subsystem level

## Structure

- `01-…md` … `0N-…md`: lecture chapters.
- `exercises/`: per-exercise prompts.
- `labs/`: long-form hands-on labs.
- `quizzes/`: knowledge checks.
- `resources.md`: external references.
